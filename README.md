# GNIOT Webathon 2K26

Welcome to the GNIOT Webathon 2K26 project repository! 
This repository contains the boilerplate template ready to be developed during the hackathon.

## Structure

- `index.html`: Main HTML file with SEO best practices and semantic structure.
- `styles.css`: Premium baseline styling featuring a dark theme, modern typography (Inter), glassmorphism, and responsive design.
- `script.js`: Core JavaScript logic with basic examples of DOM manipulation and smooth intersection observer animations.

## Getting Started

1. Open `index.html` in your browser to view the boilerplate template.
2. Edit the files to build your solution based on the challenges provided during the event.
